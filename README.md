# Show me your setup

A Instagram like SPA for sharing computer setups pictures.

## Stack

-   Node.js
-   Express
-   Postgresql
-   AWS S3
-   Vue.js

## Features

-   Visualize all the pictures posted
-   Visualize a single picture
    -   Add a comment
-   Upload a picture
    -   Add a description to the picture
    -   Add a title to the picture
-   Filter pictures by tag
